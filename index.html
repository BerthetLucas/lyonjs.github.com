Applis web : AngularJS réinvente le HTML Thierry Chatel et Matthieu Lux

PLan:

  • Présentation théorique d'Angular JS par Thierry Chatel
  • Présentation théorique de Yeoman par Matthieu Lux
  • Live coding interface CRUD from scratch par Matthieu Lux
  • Exemples de code avancés avec Thierry Chatel

AngularJS est un framework JavaScript open source, développé par Google et dont la première version stable date de juin 2012. Il est dédié à  la réalisation de la partie cliente des applications web de toutes tailles, de la petite application embarquée dans une page web jusqu'à  la grosse application monopage.

Grâce à des composants réutilisables, AngularJS transforme le HTML en un système de template déclaratif très puissant pour créer des vues dynamiques liées à un modêle de données. Les vues de l'application sont rafraïchies automatiquement à chaque changement dans le modèle.

Alors que la plupart des autres frameworks JavaScript imposent énormément de manipulations de DOM et de gestion d'évènements, AngularJS affranchit le développeur de tout ce code technique fastidieux, et lui permet de se focaliser sur le code métier de son application web.

En structurant les applications web en modules et services, avec injection automatique des dépendances, et en découplant complètement les vues du code de présentation, AngularJS rend les applications web facilement testables, à la fois par des tests unitaires de l'ensemble du code JavaScript et par l'automatisation de scénarios de tests fonctionnels.

AngularJS apporte au développement natif côté client, en HTML et JavaScript, les bonnes pratiques mises en oeuvre depuis plusieurs années dans les meilleurs frameworks serveur.
